NerpaAI - приложение для Компас-3D, включающее в себя множество функций для автоматизации работы конструктора.
Среди основных функций:
1) Адаптация сборок и деталей - автоматическое заполнение свойств подавляющего большинства деталей в
соответсвии с выработанными правилами.
2) Создание спецификаций (BOM) - создание на чертеже спецификации активной сборки.
3) Вставка шаблонов тех. требований
4) Составление таблицы гибов для труб, полученных методом холодной гибки.
5) Перевод чертежей - перевод чертежей формата .CDW с английского на русский.
6) Составление МТО для сборки с любым уровнем вложенности.
7) Создание PDF - инструмент для быстрого составления пакета документов в формате .PDF.
